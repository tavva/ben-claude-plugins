# ben-claude-plugins

My plugins* for Claude Code.

\* Yes, there's just one right now.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin install obsidian-agent-tools@ben-claude-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [obsidian-agent-tools](plugins/obsidian-agent-tools) | CDP tools for Obsidian plugin development and testing |
